# Actividad 7: Video tutorial sobre un algoritmo (Individual)

**Objetivo:**

Crear un video tutorial de máximo 5 minutos de duración en el que se explique un algoritmo de libre elección utilizando el lenguaje de programación Java.

**Descripción:**

1.  **Elección del algoritmo:** Selecciona un algoritmo de Java que consideres interesante y que pueda ser explicado en un tiempo máximo de 5 minutos.
2.  **Preparación del contenido:**
    -   **Escribe un guión:** Planifica el contenido del video, incluyendo los puntos clave que deseas explicar y el orden en que los presentarás. Asegúrate de que el guión sea claro, conciso y fácil de seguir.
    -   **Prepara ejemplos:** Utiliza ejemplos prácticos para ilustrar el funcionamiento del algoritmo. Puedes utilizar código, diagramas o animaciones para hacer que las explicaciones sean más claras y atractivas.
    -   **Graba el video:** Utiliza un software de grabación de pantalla para grabar tu video. Asegúrate de que la calidad del audio y del video sea buena y que la iluminación sea adecuada.
    -   **Edita el video:** Edita el video para eliminar errores, agregar transiciones y mejorar la calidad general del video. Puedes utilizar un software de edición de video para realizar estas tareas.
3.  **Estructura del video:**
    -   **Introducción (0:00 - 0:30):**
        -   Presenta el tema del video, el algoritmo que se explicará y los objetivos del tutorial.
    -   **Explicación del algoritmo (0:30 - 4:00):**
        -   Describe el algoritmo paso a paso, utilizando un lenguaje claro y sencillo.
        -   Utiliza ejemplos prácticos para ilustrar el funcionamiento del algoritmo.
        -   Muestra el código del algoritmo en Java y explica su funcionamiento.
    -   **Conclusión (4:00 - 4:30):**
        -   Resume los puntos clave del video.
        -   Menciona algunas aplicaciones del algoritmo.
        -   Invita a los espectadores a realizar más investigaciones sobre el tema.

**Evaluación:**

- Claridad de la explicación    30% 
- Uso de ejemplos prácticos     20% 
- Calidad del código            20% 
- Calidad del video y audio     15% 
- Estructura del video          15%
